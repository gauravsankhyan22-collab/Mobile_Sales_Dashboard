📱 Mobile Sales Dashboard | Power BI

Overview

This project is an interactive Power BI dashboard developed using a mobile sales dataset. It demonstrates the complete analytics workflow, including data cleaning, transformation, data modeling, DAX calculations, and dashboard design to generate actionable business insights.

Key Features

* Imported and cleaned raw sales data using Power Query.
* Created a custom calendar table and built relationships for efficient data modeling.
* Developed DAX measures to calculate key performance indicators (KPIs).
* Designed an interactive dashboard with KPI cards, charts, and slicers for dynamic analysis.
* Analyzed sales performance by city, mobile brand, model, time period, and payment method.
* Enabled users to explore trends and compare sales performance through interactive filtering.

Skills Demonstrated

* Power BI
* Power Query
* Data Cleaning & Transformation
* Data Modeling
* Calendar Table
* Relationship Building
* DAX
* Dashboard Design
* Data Visualization

Business Insights

The dashboard helps identify top-performing cities and brands, understand customer payment preferences, monitor sales trends over time, and track overall business performance through interactive visualizations.
